# Marist Brothers of East Asia - Professional Website

## Project Overview

A modern, professional website for the Marist Brothers of East Asia (MaristEastAsia) - a Catholic religious congregation dedicated to education and pastoral ministry serving young people across East Asia.

## Website Features

### Pages Included

1. **Home Page (index.html)** - Landing page with hero section, key activities, upcoming events, and mission statement
2. **About Us (about.html)** - Organization history, core values, leadership, and institutional details
3. **Vocation Page (vocation.html)** - Information about religious vocation and the path to becoming a Marist Brother
4. **Ministry Page (ministry.html)** - Details about both school and non-school ministries
   - School Ministries: Marist schools across East Asia
   - Non-School Ministries: Youth pastoral, social action, parish ministry, retreats
5. **News & Updates (news.html)** - Latest news with filtering and search functionality
6. **Calendar of Activities (calendar.html)** - Event calendar with school academic schedule
7. **St. Marcellin Champagnat (champagnat.html)** - Biography and legacy of the founder
8. **Our Presence (presence.html)** - Information about locations in East Asia
9. **Links Page (links.html)** - Related resources and partner websites
10. **Contact Us (contact.html)** - Contact form and office information
11. **Login Page (login.html)** - Secure login portal

## How to Use

### Opening the Website
1. Open `index.html` in any modern web browser
2. Navigate through pages using the main navigation menu
3. All links are fully functional

### Customization

#### Adding Images
Replace image URLs in the HTML files with actual images from:
- Pexels.com: https://www.pexels.com
- Unsplash.com
- Your own image hosting

Images are currently using sample URLs from Pexels.

#### Updating Content
Edit content in any HTML file - all text, headings, and descriptions are easily modifiable.

#### Contact Information
Update contact details in footer and contact page:
- Email: info@maristeastasia.org
- Phone: +65 6000 0000

#### Color Scheme
Modify colors in `assets/css/style.css` by editing the CSS variables:
```css
:root {
    --primary-color: #1e3a5f;
    --secondary-color: #c41e3a;
    --accent-color: #f5a623;
    /* ... other colors ... */
}
```
