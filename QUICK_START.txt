REPLACE IMAGES
────────────────
  Current images are from Pexels.com (free to use)
  
  To add your own images:
    1. Find image URLs in HTML files (look for src="https://")
    2. Replace with your image URLs
    3. Or save images locally in assets/images/ folder
    4. Change URLs to: ../assets/images/your-image.jpg
  
  Recommended image sizes:
    • Hero images: 1920x600px
    • Card images: 600x400px
    • Logo/icons: 200x200px
    • Team photos: 400x400px


ADD NEW PAGES
──────────────
  1. Copy an existing page (e.g., pages/about.html)
  2. Rename the copy (e.g., pages/newpage.html)
  3. Edit the content
  4. Update navigation menu:
     - In index.html, find the <nav><ul> section
     - Add: <li><a href="pages/newpage.html">Page Name</a></li>
     - Repeat for all other pages


CHANGE TEXT & CONTENT
──────────────────────
  All text is in HTML files and easily editable:
  
  • Titles: Look for <h1>, <h2>, <h3> tags
  • Paragraphs: Look for <p> tags
  • Headings: Look for <h4>, <h5> tags
  • Lists: Look for <ul> and <li> tags
  
  Simply edit the text between opening and closing tags!


MODIFY STYLES
───────────────
  File: assets/css/style.css
  
  Main sections:
    • Lines 1-15: Color variables
    • Lines 30-80: Header & navigation
    • Lines 100-140: Hero section
    • Lines 200-250: Cards and layouts
    • Lines 450-500: Responsive design
  
  Try changing values to customize!

🌐 DEPLOYMENT (PUTTING ONLINE)

STEP 1: GET HOSTING
  Choose a web hosting provider:
    • GoDaddy, Bluehost, HostGator, etc.
    • Or use cloud hosting: AWS, Google Cloud, Azure

STEP 2: UPLOAD FILES
  Using FTP or File Manager:
    1. Upload all files from MaristEastAsia folder
    2. Make sure folder structure is maintained
    3. Upload to public_html or www folder

STEP 3: SET UP DOMAIN
  Register domain name:
    • maristeastasia.org
    • maristbrotherseastasia.com
    • Or any desired name

STEP 4: TEST EVERYTHING
  1. Check all links work
  2. Test forms
  3. Verify mobile responsiveness
  4. Check image loading

STEP 5: ADD SSL (SECURITY)
  Install SSL certificate for https:// connection
  (Most hosts offer free SSL with hosting)

❓ TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

IMAGES NOT SHOWING:
  • Check image URL in HTML
  • Verify image file exists
  • Use correct file path
  • Check browser console for errors

STYLES NOT APPLYING:
  • Clear browser cache (Ctrl+Shift+Delete)
  • Check CSS file path is correct
  • Verify file is saved
  • Look for CSS syntax errors

FORMS NOT WORKING:
  • Check JavaScript is enabled
  • Verify function names match
  • Check browser console for errors
  • Test in different browser

LINKS NOT WORKING:
  • Check file paths are correct
  • Verify files exist in pages folder
  • Use relative paths (./ or ../)
  • Check for typos in file names

MOBILE MENU NOT SHOWING:
  • Clear cache
  • Check JavaScript is enabled
  • Verify hamburger.js is loaded
  • Test in different mobile browser
