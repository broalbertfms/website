Quick Start Guide

1

Start Backend Server

Open command prompt and navigate to the BACKEND folder, then run:

npm run dev

The server will start on http://localhost:5000

2

Login to Admin Panel

Use these demo credentials to login:

Email: admin@maristeastasia.org
Password: Admin@123
3

Create Your First Post

Go to News & Posts section and click "Create New Post". Use the rich text editor to format your content with:

  • Text formatting (bold, underline)
  • Headings and lists
  • Colors and fonts
  • Images and links
  • And much more!
4

Manage Events & Messages

Use the Events section to create scheduled events. Go to Contact Messages to view and respond to form submissions from visitors.

5

Customize Your Setup

Go to Settings to update your profile, change password, and configure other options. Refer to ADMIN_GUIDE.md for detailed instructions.
